Step by Step GITHUB:

  • 1. Siapkan file atau folder yang mau di upload github.
  • 2. Git Init (Inisiasi GIT, agar bisa diketahui oleh sistem bahwa dalam folder tersebut bisa dilakukan GIT routin)
  • 3. Upload File & folder

    - git remote add origin https://github.com/teguhtrian/digidev-github-repo.git
    (digunakan untuk inisialisasi awal sekali direktori dengan repo)

    - git add .

    - git commit -m "keterangan pekerjaan yg di commit"

    - git push (git push --set-upstream origin master)
